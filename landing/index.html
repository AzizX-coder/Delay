Code Engine

VS Code-Grade IDE

A full Monaco editor implementation with auto-closing tags, bracket pair colorization, Emmet HTML boilerplate, and native preview windows. Run dev servers directly inside Delay.

An 18-tool AI agent that natively interfaces with your files, notes, and boards. Multiple model integrations (OpenRouter, Groq, Claude, OpenAI) built-in.

Canvas

Miro-like Whiteboard

Infinite canvas with shape snapping, auto-routing connectors, custom fonts, advanced property panels, and complex object persistence.

Docs-Level Notes

Rich text editing with slash commands, markdown support, export capabilities, and Notion-style block formatting.

Pomodoro integration with live active countdowns directly in your navigation rail. Focus tracking, customizable intervals, and daily goals.

A native, automated download manager powered by yt-dlp. No manual dependency setup required — auto-installs and manages media instantly.
