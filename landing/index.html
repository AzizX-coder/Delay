📖 Documentation

Everything you need to get started with Delay.

Installation

  • Windows — Download .exe from GitHub Releases
  • Android / Tablet — Download .apk from the same page. Enable "Install from Unknown Sources"
  • Web — Limited features available directly in your browser

Getting Started

  • Choose your visual theme (Aura Light, Obsidian, Forest, Mocha, Ocean, Rosé)
  • Select your preferred language from 11 options
  • Customize enabled modules from the sidebar's + button

AI Configuration

AI features are completely optional. Toggle on/off in Settings → AI Intelligence. When enabled, configure your provider:

Saved Messages

Works like Telegram's Saved Messages. Send yourself text, links, or type [] your todo for quick tasks. Features include:

  • Link domain previews
  • Emoji reactions (👍 ❤️ 🔥 ⭐ 🎯 💡 ✅ 🚀)
  • Pin important messages
  • Filter by type: All, Pinned, Links, Todos
  • Search across all saved items

Vault (Secure Files)

Create folders, upload any file type. Files are stored in the app's protected sandbox on Android, inaccessible to other apps. You can share files to Telegram or any app using the native share button.

Status Dashboard

Real-time analytics of your productivity:

  • Focus time bar chart (daily minutes)
  • Tasks completed chart
  • Activity heatmap (30-day overview)
  • Current streak counter
  • Task completion rate ring
  • Data overview with horizontal bars

Privacy & Security

  • All data stored exclusively on your device (IndexedDB + localStorage)
  • No telemetry, no tracking, no analytics sent anywhere
  • AI is optional — disable it and the app works fully offline
  • Vault files protected by app sandbox on Android
  • PIN lock prevents physical access on shared devices
  • Open source — audit the code yourself on GitHub
