The Engine

16 integrated workspaces.

From notes to presentations, from code to video editing — everything runs locally with an AI Agent that controls it all.

Core

Autonomous Agent

An 18-tool AI agent with ambient context, memory recall, and a 12-turn reasoning loop. It plans, executes, and verifies across your entire workspace autonomously.

New

Smart Timer + Goals

Pomodoro timer with purpose-driven goals — set focus/break durations, daily sessions, multi-day tracking, and progress visualization.

New

Code Studio Local IDE

Full-fledged local workspace with a multi-tab Monaco editor, interactive File Tree, and a live terminal interface powered by your desktop.

New

Disk Flows

Download videos with quality selection, in-app playback, and native file management.

New

Kanban Board

Drag-and-drop columns and cards. Add custom columns, color labels, and inline editing for agile workflows.

New

Documents

Long-form rich text editor with headings, lists, code blocks, and task items — powered by TipTap.

New

Spreadsheets

Grid-based sheets with formula support (SUM), cell editing, multi-sheet tabs, and a formula bar.

New

Presentations

Create slide decks with custom backgrounds, fullscreen presentation mode, and thumbnail navigation.

New

Whiteboard

Infinite canvas with freehand pen, shapes, colors, stroke width, undo, and PNG export.

New

Voice Studio

Smart voice recording with live waveform visualization, playback, download, and organized recording history.

New

Photo Editor

Brightness, contrast, saturation, blur — plus rotation, flip, and one-click PNG export.

New

Video Editor

Import video files with timeline scrubbing, playback speed control, volume mixer, and trim markers.
