# Siyoh Platform

A premium writing and storytelling platform for writers and readers.

## Project Structure

- `siyoh-web/`: Next.js 14 Web Application (TypeScript, Tailwind CSS, Supabase).
- `siyoh app/`: Flutter Mobile and Desktop Application.
- `design/`: Design assets and original HTML/CSS prototypes.

## Web Application Setup

1. `cd siyoh-web`
2. `npm install`
3. `cp .env.local.example .env.local` (and fill in Supabase credentials)
4. `npm run dev`

## Mobile Application Setup

1. `cd "siyoh app"`
2. `flutter pub get`
3. `flutter run`

## Features

- **Read**: Long-form essays and fiction with a focus on typography and readability.
- **Listen**: Integrated audio player for voice essays and field recordings.
- **Write**: Minimalist writing interface with draft saving and one-click publishing.
- **Admin**: Comprehensive dashboard for platform moderation and statistics.
